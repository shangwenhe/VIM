# 忘记MYsql的密码

### 1、修改配置文件

```
vim /etc/my.cnf
skip-grant-tables
```

### 2、重启mysql

```
systemctl restart mysqld
```



### 3、更新密码字段

```
UPDATE mysql.user SET authentication_string = password ( ‘new-password’ ) WHERE User = ‘root’ ;  
```

### 4、重启mysql 并登录
